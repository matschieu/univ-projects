# Running the project

* Build app using Maven:
```
mvn clean install
```

* Running the app using Maven:
```
mvn exec:java -Dexec.mainClass="com.github.matschieu.labyrinthe.Labyrinthe"
```

* Running the app using java command:
```
java -cp target/classes/ com.github.matschieu.labyrinthe.Labyrinthe
```

* Or simply execute the built executable jar located in target 
```
java -jar target/univ-projects-labyrinthe-1.0.0-SNAPSHOT-jar-with-dependencies.jar
```
